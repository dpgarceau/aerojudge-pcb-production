# Single-Board Design Files

This folder will contain the final single-board design created by the professional PCB design firm.

## Expected deliverables:
- Complete KiCad project files (or equivalent)
- Manufacturing-ready Gerber files
- Bill of Materials (BOM) with JLCPCB part numbers
- Pick-and-place files for automated assembly
- Design documentation and notes

## Verification checklist
Design must pass all milestone reviews per the project recovery plan before being accepted.
